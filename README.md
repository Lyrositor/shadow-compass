# Shadow Compass

**Shadow Compass** is a data exploration tool for the game **[Sultan's Game](https://store.steampowered.com/app/3117820/Sultans_Game/)**. It parses game files to extract game data and writes it to an HTML format.

**License:** [Creative Commons CC0 1.0 Universal](LICENSE.md)

The tool is under heavy development and is not yet ready for widespread use.
